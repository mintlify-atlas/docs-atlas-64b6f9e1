SVAR Svelte Gantt is available in both open-source (MIT) and [PRO editions](https://svar.dev/svelte/gantt/#pro). The PRO edition unlocks a suite of advanced project management capabilities built on top of the core widget.

## Export Capabilities

The PRO edition provides two complementary export pathways:

- **Server-side export** (PDF, PNG, XLSX) — the widget serializes its current state and sends it to the hosted SVAR export service (`https://export.svar.dev/gantt/<version>`). The service renders the chart and returns the binary file.
- **Client-side MS Project import/export** — XML files are read and written entirely in the browser using the `import-data` and `export-data` actions with `format: "mspx"`. No server round-trip is required.

See the [Export page](/pro/export) for full configuration details.
